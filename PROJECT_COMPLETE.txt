🚀 HOW TO START SPARK (5 Steps)

1. Terminal 1: python run_server.py
   (Backend starts on http://127.0.0.1:8000)

2. Terminal 2: npm run dev
   (Frontend starts on http://localhost:5173)

3. Browser: http://localhost:5173
   (Opens SPARK HUD interface)

4. Say: "Hey Spark"
   (Wake word detected, command bar opens)

5. Speak a command: "Create a task called shopping"
   (STT transcribes → LLM processes → TTS responds)
